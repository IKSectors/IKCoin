# IKCoin
IKCoin Cryptocurrency Project

IKCoin started as a project development on crypto tokens created on the Ethereum ecosystem.
It was first deployed on the Ropsten Testnet. It will further be migrated to Goeli Testnet, then
finally go live on the proper ethereum's network. 

Further works will see IKCoin evolve from crypto tokens to a full coin with its own blockchain
system running its own network.
